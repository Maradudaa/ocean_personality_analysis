# ocean_personality_analysis

Проект посвящён комплексному статистическому анализу данных психометрического теста **Big Five (OCEAN)**.  
Цель проекта — продемонстрировать практическое применение методов анализа данных и математической статистики на реальном наборе данных: от предобработки и агрегации признаков до оценки распределений и проверки статистических гипотез.

Проект ориентирован на анализ личностного фактора **Extraversion (экстраверсия)** и показывает полный аналитический пайплайн.

# Используемые данные

Источник данных:  
**OpenPsychometrics — “OCEAN Five Factor Personality Test Responses”**
Датасет доступен на платформе Kaggle: 
https://www.kaggle.com/datasets/lucasgreenwell/ocean-five-factor-personality-test-responses

Характеристики датасета:
- ~19 700 наблюдений
- 57 признаков
- Ответы респондентов на 50 утверждений теста OCEAN
- Шкала ответов: от 1 до 5 (Likert scale)

Данные охватывают пять личностных факторов:
- Extraversion
- Neuroticism
- Agreeableness
- Conscientiousness
- Openness

# Этапы анализа

## 1. Предобработка данных
- Анализ структуры и качества данных
- Реверсирование шкал для обратных утверждений
- Агрегация ответов по факторам OCEAN
- Формирование итоговых числовых признаков

## 2. Разведочный анализ данных (EDA)
- Построение вариационных рядов
- Гистограммы распределений
- Эмпирическая функция распределения (ECDF)
- Анализ формы распределения и симметрии

## 3. Описательная статистика
- Выборочное среднее
- Медиана и мода
- Дисперсия и стандартное отклонение
- Коэффициент вариации

## 4. Оценка параметров распределения
- Точечные оценки (метод моментов, метод максимального правдоподобия)
- Доверительные интервалы для среднего и дисперсии
- Анализ свойств оценок (несмещённость, состоятельность, эффективность)

## 5. Оценка плотностей и распределений
- Ядерная оценка плотности (KDE)
- Сравнение эмпирического распределения с нормальным
- Анализ отклонений от теоретической модели

## 6. Проверка статистических гипотез
- Одновыборочный t-тест
- Двухвыборочный t-тест
- Корреляционный анализ (коэффициент Пирсона)
- Оценка силы эффекта и мощности тестов

## 7. Идентификация распределения
- Критерий согласия Пирсона (χ²)
- Анализ ограничений применения критериев при большом объёме выборки

# Основные результаты
- Распределение показателя экстраверсии близко к нормальному  
- Среднее значение экстраверсии статистически не отличается от нейтрального значения шкалы  
- Обнаружена слабая, но статистически значимая корреляция между экстраверсией и открытостью опыту  
- Показано влияние объёма выборки на чувствительность статистических критериев
